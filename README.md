# CHIP-8 emulator

<p align="center">
<img src="doc/c-chip8-00.png" alt="c-chip8 title image."/>
</p>

A simple and functional CHIP-8 emulator written in C and SDL3.


## Installation

1. Install dependencies:
    - Debian Linux: `sudo apt install build-essential libsdl3-dev`
2. Clone the repository.
3. Build the CHIP-8 emulator from the source code using the `make` command.

## Usage

Run the emulator using: `./chip8 <path/to/rom>`
